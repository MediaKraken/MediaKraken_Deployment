This remote is available from Conrad Electronics (www.conrad.de, order number 35 01 57 22). Before you can use this remote, you have to enter a device-specific code. I listed the device-codes I used to have all buttons available (except "record" for the TV-Device). For AUX and SAT you can also use VCR-device-codes, but not all work. TV for example only accepts TV-device-codes. In the picture above you can see how I named the buttons, and for each device I made a config file, named "RM200-device-name". If you want all four devices, just type "cat RM200-AUX RM200-TV RM200-VCR RM200-SAT > lircd.conf".
